Secret Labs
  • Microcontrôleur : STM32F405RG 32 bits à architecture Cortex-M4 cadencé à 168 MHz.
  • RAM : 100 KB (164+)*.
  • Flash : 384 KB (1408KB)*.
  • Port Ethernet : 10Mbps (Wifi:802.11b/g/n)*.
  • E/S numériques : 14
  • Entrées analogiques : 6
  • IDE : Microsoft Visual Studio
  • Framework : .NETMF 4.3
  • Langages de programmation : C#, VB
